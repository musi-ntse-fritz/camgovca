Opérations sur les certificats

Installation de l'utilitaire de gestion des certificats

Installation de l'utilitaire de gestion des certificats

Cet utilitaire, une fois installé, vous permet de procéder à toutes les opérations de gestion d'un certificat électronique. Télécharger et installer l'utilitaire.

Émission de votre certificat

Émission de votre certificat

Pour émettre un nouveau certificat, veuillez saisir le numéro d'enregistrement du certificat se trouvant sur le recépissé que vous avez reçu de l'opérateur de l'Autorité d'Enregistrement, puis cliquez sur le bouton "émettre votre nouveau certificat" .

Reémission de votre certificat

Reémission de votre certificat

En cas de perte ou de corruption de votre certificat, vous pouvez en obtenir un nouveau. Suivez la même procédure que celle utilisée pour l'émission de votre nouveau certificat.

Renouvellement de votre certificat

Renouvellement de votre certificat

Au plus 30 jours avant sa date d'expiration, vous pouvez étendre la date de validité de votre certificat. Pour le faire, saisissez votre code obtenu auprès de votre autorité d'enregistrement, Une fois le code vérifié, veuillez sélectionner votre certificat, entrez votre mot de passe et validez. La date d'expiration sera repoussée d'un an.

Copie de votre certificat

Copie de votre certificat

Vous pouvez copier le certificat sauvegardé sur le disque dur, une clé USB ou une carte à puce et l'enregistrer sur un autre support de stockage.
Vérification du mot de passe de votre certificat

Vérification du mot de passe de votre certificat

Cette fonction permet de vérifier le mot de passe du certificat.
Vérification du numéro d'identification de votre certificat

Vérification du numéro d'identification de votre certificat

Cette fonction permet de vérifier le numéro d'enregistrement de votre certificat. Ce numéro correspond à votre numéro de carte nationale d'identité, le numéro du registre de commerce de votre entreprise ou le code budgétaire de votre Administration utilisé lors de votre identification auprès de l'Autorité d'Enregistrement.
Changement du mot de passe du certificat

Changement du mot de passe du certificat

Cette fonction vous permet de changer le mot de passe de votre certificat.
Suspension temporaire de votre certificat

Suspension temporaire de votre certificat

Pour la suspension de votre certificat, veuillez vous rendre auprès de votre Autorité d'Enregistrement de rattachement pour vous enquérir des modalités. Après suspension, votre certificat ne sera pas valide pour une durée de six (06) mois.

Suppression de votre certificat

Suppression de votre certificat

Cette fonction vous permet de supprimer votre certificat de votre support de stockage.
Révocation de votre certificat

Révocation de votre certificat

Pour la révocation de votre vertificat, veuillez vous rendre auprès de l'Autorité d'Enregistrement Centrale de l'ANTIC pour vous enquérir des modalités.
